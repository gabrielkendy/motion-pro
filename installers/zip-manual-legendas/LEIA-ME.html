🚀 Instalação automática

  1. Feche o Adobe Premiere ProSe estiver aberto, feche antes.
  2. Duplo-clique em INSTALAR.batJanela preta de terminal vai abrir. É normal, não é vírus.
  3. Aguarde "INSTALADO" (~30s)O script copia 549 mogrts + configura o registro do Windows.
  4. Pergunta se quer abrir PremiereDigite S + Enter, ou abra manualmente.
  5. No Premiere: Janela → Extensões → MotionPro LegendasFaça login (ou crie conta — 14 dias grátis).
✓ Sem aviso do Windows SmartScreen — .bat é script, não binário executável.
