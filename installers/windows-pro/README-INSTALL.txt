   MotionVault · Antes de Instalar
════════════════════════════════════════════════════════════════

REQUISITOS:
• Windows 10 ou 11
• Adobe Premiere Pro CC 2019 ou superior
• Conexão com internet pra ativação

ANTES DE COMEÇAR:
1. Feche o Adobe Premiere Pro (e After Effects, se estiver aberto)
2. Tenha em mãos o e-mail + senha temporária que recebeu
   após sua compra. (Cheque também a caixa de spam.)

O QUE ESTE INSTALADOR FAZ:
• Copia o plugin para a pasta correta do Adobe CEP
• Habilita o modo de extensões personalizadas no registro
• Limpa caches antigos do Adobe pra evitar conflito
• Cria entrada no "Adicionar/Remover Programas" do Windows
• Permite desinstalar limpo a qualquer momento

DEPOIS DE INSTALAR:
1. Abra o Premiere Pro
2. Menu Janela → Extensões → MotionVault
3. Faça login com suas credenciais
4. Pronto! 7.906 templates disponíveis no painel.

PROBLEMAS COMUNS:
• "Plugin não aparece no menu": Reinicie o Premiere completamente
• "Login falhou": Confirme se digitou o e-mail correto, sem espaços
• "Senha inválida": Use o link 'Esqueci minha senha' no site
