## 6 · Como ativar OAuth (passos manuais)

### Google
1. https://console.cloud.google.com/apis/credentials → "Create Credentials" → OAuth client ID
2. Application type: Web application
3. Authorized redirect URI: `https://motionpro.vercel.app/v1/oauth/google/callback`
4. Copiar Client ID + Secret → env vars
5. Vercel → Settings → Environment Variables → Redeploy

### GitHub
1. https://github.com/settings/developers → "New OAuth App"
2. Homepage URL: `https://motionpro.vercel.app`
3. Authorization callback URL: `https://motionpro.vercel.app/v1/oauth/github/callback`
4. Copiar Client ID + Secret → env vars
5. Vercel → Redeploy

### Magic link
Funciona automaticamente assim que Resend estiver configurado (já está).
